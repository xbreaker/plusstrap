Introducing Plusstrap.

Need reasons to love Plusstrap? Look no further.

By nerd, for nerds.

Forked from Twitter Bootstrap, built by @xbreaker, Plusstrap utilizes LESS CSS, is compiled via Node, and is managed through GitHub to help nerds do awesome stuff on the web.

Made for everyone.

Plusstrap was made to not only look and behave great in the latest desktop browsers (as well as IE7!), but in tablet and smartphone browsers via responsive CSS as well.

Packed with features.

A 12-column responsive grid, dozens of components, javascript plugins, typography, and form controls.
